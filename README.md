# Ruby on Nails

What we have here ladies is a small framework called Ruby on Nails,
which is intened for learning 

## Installation

Add this line to your application's Gemfile:

    gem 'nails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install nails
